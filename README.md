# Atlas Mining Corp. — Website

Landing page for Atlas Mining Corp., the AI Geologist. Uses machine learning to identify high-probability mineral deposits at a fraction of traditional exploration cost.

## Local Development

```bash
python3 -m http.server 8090
```

Then open [http://localhost:8090](http://localhost:8090).

## Deployment

Enable GitHub Pages in repo settings (Settings > Pages > Source: `main` branch) to host at `atlas-mining.github.io/website`, or point a custom domain.
